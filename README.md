# 安装
[Windows下载地址](https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe)

也支持其他平台。
# 创建项目
```shell
cargo new 项目名称
```
如果路径已经存在，使用如下命令即可。
```shell
cargo init 项目名称
```
